## 📊 inDriver Trip Analysis Dashboard

### 📌 Project Overview
This project analyzes ride-hailing data to evaluate trip performance, customer behavior, and operational efficiency.

### 🛠 Tools Used
- Power BI  
- Microsoft Excel  

### 📈 Key Insights
- 85% of trips were completed, while 15% were canceled  
- Average customer rating: 3.4  
- Average trip duration: 28 minutes  

### 📊 Dashboard Features
- Trips by region  
- Driver performance  
- Payment methods  
- Trip status and cancellation reasons
